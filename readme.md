->A Complete Full Stack Blog Application where a User (public/private) can add,read, update, and delete blog posts.
->React for Frontend and Express (Nodejs) for backend
->Mongodb (Mongoose) for database
->Authentication service for user with encrypted password storage facility using salting and hashing
->For image upload, used both local (Multer) and remote (appwrite) service.

->Used the Services of React Hook Form (for managing all the state related work of a html form in react seamlessly) and Tiny MCE Text Editor.

->Implemented the following concepts in building this application:-
--React Redux, React Router along with hooks like useState,useRef,useNavigate, etc.. and other basic functionalities of React (Frontend)
--Server creation using Express, GET and POST Routes handling, Middlewares and body-parser setup, Multer file storage and unlinking (Backend)
--Creation of Schemas | Collections | Documents , Read | Write | Delete operations, Embedding documents using references (Database)
--Salting and Hashing for storing password of user in Authentication setup of both login and signup (Authentication)

->Also gave the user the option to post public posts also without logging (i.e. public posts) and separated private from public posts.

->User suggestions and improvements are always welcomed!!
